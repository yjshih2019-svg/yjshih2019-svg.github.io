README

This project studies stock trends and financial indicators. The strategy combines revenue growth screening and Hurst exponent analysis.

  • Revenue growth screening
  • Trend / mean-reversion detection
  • Financial statement evaluation

Sample Code


import pandas as pd

def hurst_signal(h):
    if h > 0.5:
        return "Trending"
    elif h < 0.5:
        return "Mean Reverting"
    else:
        return "Random Walk"

print(hurst_signal(0.62))
